# Customer Behaviour Analysis Dashboard


## 📌 Project Overview

This project focuses on analyzing customer churn data to identify the main reasons why customers leave a telecom company.
The project includes data cleaning, SQL analysis, Python-based EDA, and an interactive Power BI dashboard for visualization.

## 🔍 Key Insights

* Customers with month-to-month contracts had higher churn rates.
* Electronic check payment users showed more churn behavior.
* Customers with longer tenure were less likely to churn.
* Fiber optic internet users had comparatively higher churn.
